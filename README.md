### 자료조사
* 1차 회의 : 평소 생활 속에서의 니즈를 찾으려고 노력함 -> 전세계가 재난상황에 처한 이후 집에 있는 시간이 많아짐에 따라, 집 안에서의 불편함을 해소할 수 있는 아이디어를 생각함.
